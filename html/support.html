Common questions

  • Is it possible to use the game assets in a (commercial) project?

    All game assets found on the asset page are public domain CC0 licensed, see the included license file or refer to the game asset page for full details.

  • How can I use the game assets in my game engine?

    We unfortunately can't offer help when importing our game assets in game engines, please refer to the community or support of the game engine.

  • Should attribution be given when the game assets are used?

    Attribution is not required but if you choose to credit refer to 'Kenney', 'Kenney.nl' or 'www.kenney.nl'. Do not use our logo as it is reserved for projects by our studio.

  • I'd like to submit an idea for game assets

    Ideas are more than welcome, you can tweet your idea to us for a chance of them being made.

  • I've lost the download link or e-mail for a purchased product

    You can recover lost downloads or purchases through Itch.io support, Steam support or Epic Games Store support.
